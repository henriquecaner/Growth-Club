Quando o vetor oficial sair do Figma, **mantenha os mesmos nomes de arquivo** (`logo-white.svg`, `logo-black.svg`, `logo-mark.svg`, `logo-mark-white.svg`). Todos os HTMLs em `preview/`, `templates/`, e `ui_kits/` referenciam esses paths — a substituição é zero-touch.
